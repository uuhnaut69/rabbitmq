Using Kafka if need: 
- Time travel/durable/commit log
- Many consumers for the same message
- High throughput
- Stream processing
- Replicability
- High availability
- Message order

Using RabbitMq if need:
- flexible routing
- Priority Queue
- A standard protocol message queue
